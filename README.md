Aquí puedes ver algunos de los desarrollos más interesantes que he realizado durante mi trayectoria en la UJA:

   - **Tecnologías:** `Java`

2. **[OpenGL-Graphics-Visualization](https://github.com/ivaanms10/OpenGL-Graphics-Visualization)**
   - **Descripción:** Proyecto enfocado en la computación gráfica y renderizado en 3D para la visualización de entornos geométricos.

3. **[Paso-Mensaje-JMS](https://github.com/ivaanms10/Paso-Mensaje-JMS)**
   - **Descripción:** Implementación de arquitectura de software para sistemas distribuidos utilizando mensajería asíncrona mediante el estándar Java Message Service (JMS).
   - **Tecnologías:** `Java`

   - **Descripción:** Aplicación web interactiva simulando una plataforma de pedidos para una pizzería universitaria.
   - **Tecnologías:** `HTML`, `CSS`, `JavaScript`
